# 记忆缝线

设定有点类似《心理测量者》，不过质量就完全比不上了。

第一集刚来就跳跃剧情，出场的人物和相关世界观设定也没有合理地解释，看得一脸懵逼。大量的高光都打在男女二人无聊的拌嘴上了，这一点上有些类似《侦探已死》，但显然也比不上后者。

科幻设定也非常小气。都已经脑机直连AI的科幻世界了，日本人还在用纸币购物，杀人犯还在平平无奇的用小刀杀人，甚至还没《柯南》来得有新意。同时他们对欧美文化的仰慕也几乎要溢出屏幕，在2026年中美贸易战、中日关系紧张的时间点回头来评价这种崇洋媚外的心理，也让人挺难绷的。

说来搞笑，这部番刚开头的弹幕还在刷夫妻档，我还没来得及去搜索香菜结婚的消息呢，然后很快又从弹幕里得知香菜已经离婚了，令我哭笑不得。香菜演的女主也丝毫没有吸引我，甚至远远不如短暂登场的东山来的可爱。

画面质量还可以，音乐不咋地，叙事很糟糕，综合评分1星。

# 志愿军：雄兵出击

如果要用严格的标准来评价这部电影的“正确性”，我的答案是：不及格。

对于战场的表现有很多不合理的地方，例如战场上士兵密度过大、炮火密度过大，例如站着开枪，例如解放军的衣服太新太干净、武器装备太豪华等等。跟西方的一些优秀战争电影，例如《拯救大兵瑞恩》、《我们的父辈》相比，差距还是很明显的。

对于敌人的表现也过于主观和粗暴，把美军拍得跟抗日大剧中的日本鬼子似的简单无脑、邪恶嘴脸。实际上虽然麦克阿瑟将军狂妄、美帝邪恶，但美军的战斗力确实很强，是值得尊敬的对手。

但如果把大脑放在一边，用屁股稳坐在“中国人”的立场上，像好莱坞电影或者爽文小说一样来看本作，我觉得还是不错的，情感渲染还是比较到位的。

还有一点想吐槽的是，有几位比较“现代化”的演员，在演这部戏的时候感觉有点不是很合适，我光是看着他们的脸就感觉出戏了，笑死。

还有一点想吐槽的是，这次我是在优酷上看的，明明我已经是会员了还要看各种广告，有时候真是气到无语，恨不得把这些大厂产品经理的心肝挖出来看看到底是不是黑的。

# 长津湖

开头男主抱着骨灰罐回家探亲的场景，很有武侠小说那种把神秘的江湖缓缓展开的风格；中间的几场战斗虽然不尽完美，但也算得上斗智斗勇。看到这里我觉得可以是一部4星或者5星的好作品的。

可是最后1/3又回到了老套的抗日神剧环节，过于刻意（或者说偷懒）地丑化敌方和神化己方，实在是降智。

豆瓣网友替我精确描述了这一问题：“两场轰轰烈烈的战斗，一场模模糊糊的战役”。

立意方面挺好的，特别是最后雷公死前的“懦弱”表现，点出了“英雄只是一个个平凡的战士”这一主题，挺值得回味的。虽然对宏观战役描写得很模糊，但是对微观战士个体的情感思想表达得还是挺到位的。

总体来说比我刚刚看的《志愿军：雄兵出击》更好一些，但是也好得有限，我的综合评分是3+或者4-星。

让我有点吃惊的是易烊千玺的演技。不知为何，虽然我已经知道易烊千玺已经演过不少的电影，这些电影中不乏佳作，可我仍然习惯性地把他看作一个花瓶偶像。而在这部电影中，他的表现也是可以算得上“优秀”的，他再一次地证明了自己，我也不得不再一次承认他的成绩——大概在年轻一辈的演员中他已经算得上是最好的一批了吧。

# 碧蓝之海

故事设定主角们属于一个潜水社团，但是在剧中，潜水所占的戏份有点太少了。如果潜水相关的知识再多一点，多少还能当个科普番，搞笑之余收获增长一些见闻，可惜了。

故事的核心还是搞笑。我能感觉得出来本作已经很用力在搞笑了，但我始终觉得有点笑不太出来。因为感觉有些低俗，例如在女人面前光屁股、喝酒宿醉、上课不认真、考试作弊……也许很多人会说“这就是青春啊”，但是很抱歉，作为985毕业姑且算是个好学生的我，并不觉得这样放纵自己就等于“青春”。

哪怕只说搞笑的部分，我也觉得没什么亮点——无厘头的部分太多了，颜艺也太多了，有种为了搞笑而搞笑的刻意感。

妹子们挺可爱，稍有暧昧的校园恋爱日常也算是日剧标准配置。

音乐很不错，尤其是OP，在我看这部番之前、很早就在我的歌单里了。

总体评分中规中矩，3星。

# 忍者与杀手二人组的日常生活

歪，三观就是歪，没有任何辩解的余地。

本作是很典型的女频凤傲天配置，一个无敌的腹黑女主带一个天然无脑同时幸运max的女二，跟其他的女孩战斗，轻百合，总体来说设定没什么问题。

但最大的问题，同时也是一个很隐蔽的问题是，女主杀人太过于轻易了。

支持女主的人会说，女主设定就是个杀手，不能以常人的三观去评价，杀人也是为了保护女二不被追杀，不该在二次元的世界追求真实感。

但实际上，二次元的虚构也是有底线的，必须要遵守道德和法律的底线。比如后宫番，无论再怎么暧昧、再怎么暗示，也不能公然宣布所有后宫都是老婆，“正宫”只能有一个；比如异世界龙傲天，也不能随随便便杀人，就算是很生硬地编也要编一个那人该死的理由。但是本作很明显地逾越了这条底线，杀人不需要任何理由，事后也没有任何忏悔。

算是我的偏见吧，在我印象中，女频小说大多都是像这样缺乏正义感、不讲底线的。

此外，当我发现女主的演员是花泽香菜的时候，这种厌恶感突然又被放大了。也不知道从什么时候开始，我对香菜已经粉转黑了，就感觉她的本音其实很难听，如果不刻意夹的话。还有如果我没记错的话，似乎近两年她也没有出演过正经番剧的主角了。

除了上述这个三观问题之外，本作的制作还是很精良的。采用了《物语》风格的分镜和美术风格，故事也讲得比较轻松搞怪，如果能忍受三观问题的话，还是得一看的。综合评分3星。

# 疯狂动物城2

最近在网络上关于“斩杀线”的话题得到了过分的曝光和关注，以至于一些关于本片的影评也以“灵视提高”的角度进行分析。但是我在看过之后，觉得并没有那么夸张。诚然，本片确实反应了西方社会的许多问题，但我并不觉得那些黑暗面就是本片的全部，尽管你可以嘲讽，但勇气与希望依然同样存在并贯穿全片故事脉络。综合来看我觉得本片也就是一个普通的现代社会侦探冒险故事，有邪恶也有正义，任何文明包括我们自己都能拍出类似的故事来，当普通的爆米花电影看就完了，没必要上纲上线。

故事是很经典的好莱坞风格，制作水平也是世界一流水平，没什么好说的。
